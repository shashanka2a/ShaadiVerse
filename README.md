# Shaadiverse - Next.js 15 App

A production-ready Next.js 15 application for Shaadiverse, India's trusted budget wedding directory.

## Features

- ✅ Next.js 15 with App Router
- ✅ Tailwind CSS v3
- ✅ TypeScript
- ✅ SEO meta tags (Open Graph, Twitter Cards)
- ✅ Lucide React icons
- ✅ Responsive design
- ✅ Optimized images with Next.js Image component

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```
